----------------- devops ----------------------------------

1. docker image create :-
     ✅ docker build . -t  express_server 

    2. docker run -p 8080:3000 express-server
     ✅ port maping karata hai

3. docker compose up 
✅ Image build karti hai (agar zarurat ho)
✅ Container create karti hai
✅ Container start karti hai
✅ Network create karti hai (agar nahi bana ho)

4. docker compose down Ye command:
✅ Container stop karti hai
✅ Container remove karti hai
✅ Compose network remove karti hai
